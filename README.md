# Dark Purple Accent (Iris Spring) Theme for Mailspring

A simple edit of the [Mailspring theme starter](https://github.com/Foundry376/Mailspring-Theme-Starter) to change the accent colour to purple to match Windows accent colour - Iris Spring.
